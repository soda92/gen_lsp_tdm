# demo

## setup
install `gen-lsp-tdm` from [PYPI](pypi), and [make][make]/[cmake][cmake] from scoop, and tdm-gcc from [official website][tdm]

## usage
run `tdm_cmake_init`
then run `make`

## expected result

LSP correctly shown, no errors in VSCode problem panel
